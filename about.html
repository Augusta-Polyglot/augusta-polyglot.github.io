The Augusta Polyglot programming group was started to fill the need for a programming community in the Central Maine area. Two local programmers wanted to get together to learn more about their craft and the rest is history.

The group is designed to be hands on. Each meeting we pick a topic or a set of programming puzzles and work together to learn new skills. We leave no one behind and no question is is left unanswered. We cover all languages and programming paradigms.

Ployglot programming - a definition

Type polyglot definition into google and you get:

knowing or using several languages.

We wanted to stretch our programming skills so we decided to include all types of languages in our group activities. This way we are exposed to new types of the programming techniques and patterns.
